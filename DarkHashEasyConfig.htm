DarkHash Easy-Config v1.4
Switch up your mining!
-
DarkHash is an automated mining assistant for use with the Dark Fiber Mines pool.
It allows you to mine a multitude of coins between given intervals with automatic stop-and-start functions.
This version of DarkHash Easy-Config is for DarkHash v1.3 and v1.4
-
This one-time tool allows you to easily create your initial DarkHash configuration without the need to individually modify the darkhash.bat settings.
Simply complete the below tables and copy/paste the output beneath into config.bat.
-
Standard Miner(s):
You must have the following miner(s) in your DarkHash main directory:
miner.exe (Gminer)
-
Additional Miner(s):
You may require the following miner(s) in your DarkHash main directory:
lolMiner.exe (lolMiner) - if mining BEAM
miniZ.exe (MiniZ) - if mining BEAM with 4GB NVIDIA card(s)
teamredminer.exe (Team Red Miner) - if mining Ravencoin using AMD GPU(s)
t-rex.exe (T-Rex) - if mining Ravencoin using NVIDIA GPU(s)
