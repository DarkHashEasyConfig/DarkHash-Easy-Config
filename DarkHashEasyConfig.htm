DarkHash Easy-Config
DarkHash Easy-Config v1.0
Switch up your mining!
-
DarkHash is an automated mining assistant for use with the Dark Fiber Mines pool.
It allows you to mine a multitude of coins between given intervals with automatic stop-and-start functions.
-
This one-time tool allows you to easily create your initial DarkHash configuration without the need to individually modify the darkhash.bat settings.
Simply complete the below tables and copy/paste the output beneath into darkhash.bat.
-
Remember, you must have your miner.exe and t-rex.exe files in the DarkHash-main directory!
Once configured, execute darkhash.bat to start mining.
